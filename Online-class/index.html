录制网课视频的千层套路

网课录制视频的详细心得总结。包括录制前的准备（硬件、录屏软件、后期软件）、具体录制方案和效果展示。

由于疫情的影响，最近全国从小学到大学没办法线下开学，所以老师们纷纷开启网课模式。但是无论直播还是录制视频，都会有这样那样的困难。因为 Jackie 之前有录制视频和做直播辅导课的经历，对录制视频和直播教学的工具做过调研，所以这里分享一下自己的心得，希望能对老师们有所帮助。对于直播课，各个学校应该都有硬性规定使用的直播平台，所以本文主要面向需要录播网课的老师，讲解视频的录制和后期处理（后期处理部分只做简介）。本文提到的和板书书写和音频相关的部分，对上直播课的老师们也适用。

录制前的准备

硬件

因为很多学科有板书的需求，所以用来手写的设备是必要的。可以采用以下搭配：

  • 电脑 + 数位板（手写板）
  • 平板 + 手写笔，如 ipad + apple pencil 或者 surface + surface pen

如果对录制的声音有要求，需要配一个好一点的耳麦（甚至声卡）。ipad 和 macbook 自带的麦克风效果挺好的，对录制网课来说完全足够。直接用电脑接耳机来录的话容易有喷麦声，简易的解决方案用湿纸巾包在麦克风上防喷麦。也可以买专业的麦克风（只是录制不是直播的话可以买录音笔）和防风罩。

录制软件

录制时，需要两类软件：

  • 录屏软件。电脑端推荐 Bandicam，OBS，EV 录屏等。ipad 上可以用系统自带的录屏功能。
  • 用来写板书的软件。如 PowerPoint，可以在PPT上写字；Acrobat，在 PDF 上写字。但这两个软件对笔迹的优化做得不太好，所以不太推荐。Windows 平台推荐 drawboard pdf，Macbook 或 ipad 上推荐 notability，可以在 pdf 上书写，对笔迹的优化都做得不错。

还有二合一的方案，同时可以写板书和录屏的工具，classin。classin 是一款用于在线直播的软件，我自己做线上直播辅导课就是用的 classin，非常好用（新东方的在线直播，东方优播也是用的classin）。classin 用于线上直播是收费的（在疫情期间对所有公立学校免费），但在备课教室录课是免费的，可以在 ppt，pdf，word 等所有通用格式的课件上书写，也可以直接在备课教室的黑板上书写，笔迹的优化也很棒。当然 classin 做得最好的其实是它的直播互动，一对多实时在线视频，能保证学生的听课效果，这也是我用 classin 而不是用其他免费软件进行线上直播辅导的原因。

后期软件

一次录制视频可能不成功，这时候除了重新录制一遍，还可以通过简单的后期剪辑软件来挽救。比如用会声会影、PR 等软件进行视频的剪辑，拼接，添加背景音乐等。

另外，如果需要加字幕，可以直接用 PR，也可以用专门加字幕的软件提高效率，比如用 Arctime 生成字幕再导入 PR.

如果需要对录制的人声进行处理，如修音，变得更好听，或者降噪（降低环境里的其他声音），可以用 AU 等软件。另外视频声音的响度是有国际标准的，在新版的 PR 软件渲染时设置为标准响度，听起来会比较舒服。我自己录的视频没处理响度，毕竟只是业余剪视频，电脑也带不动太耗性能的。

视频录制完，或者剪辑好渲染导出后，需要进行压制以便于上传。压制视频，就是在尽量保证视频清晰度损失不大的前提下，将原来的视频变成较小的文件。比如我在 ipad 上录出来的 500MB 的视频，压制以后就只有 30MB，方便上传到授课平台。压制视频的软件推荐B站UP主们必备的小丸工具箱，它的压制算法做得非常好，最后会生成 mp4 格式的文件。

具体录制方案和效果展示

以下方案是我使用过的方案，会给出具体的效果图片和视频链接，都是个人推荐使用的方案。

方案一：ipad + apple pencil + notability + ipad 自带录屏

上图是我在 notability 里导入 pdf 文件以后在上面书写的效果。图中的 pdf 是我用 LaTeX 排版的寿险精算 1 的讲义，做 MATH 273 辅导的时候专门写的。（只提供给参加辅导的同学，不对外提供）

apple pencil 的书写很精准，比用两三百块的手写板的效果好。不过用 ipad 自带的录屏功能生成的视频体积较大，需要用小丸工具箱压制一遍（很容易操作）。

去年为了测试 ipad 录制视频的整个流程，在 B 站发了一个试录制的视频钱是如何生钱的？CT1 金融数学 第一课 货币的时间价值（方案为：ipad + apple pencil + marginnote + ipad 自带录屏）

在里面截了一张图，如下。

因为 ipad 的屏幕长宽比不是刚好 16 : 9 ，所以用 720P 或者 1080P 这样的标准格式剪辑后，最终的视频会有黑边。

另外，发弹幕的朋友看到可以联系我把这五毛钱的工钱领了（手动狗头）。

方案二：ipad + apple pencil + classin + 云端录课

这里因为我用 ipad 直接截的图，所以上下方有黑边。实际上 classin 云端录课生成的 mp4 文件没有黑边，直接是 720P（1280×720）的，直接传到录课视频的平台里会全屏铺满，没有黑边。图中是我直接在 classin 的黑板上书写，黑板上可以同时写50页，完全足够了，不够用的话可以删除掉前面写的板书。相当于一个教室有了五十张黑板，擦黑板的工夫节约了很多，很赞了。classin 在黑板上直接写的板书可以保存为 classin 自行研发的 edb 格式。classin 里在 pdf 上书写的示例可以看方案三。

classin 的云端录课是按照使用情况收费的，但是疫情期间对公立院校（包括小学，中学，大学）免费。老师们感兴趣不妨去申请一下，需要提供姓名，电话和公立学校证明，办学资质等材料。

方案三（强烈推荐）：电脑 + 数位板 + classin + 本地录课

classin 的本地录课功能对所有账号都是免费的，用任意手机号注册，点进备课教室进行录课即可。

用这个方案录制了 【CM1-金融数学（利息理论）】真题题型精讲视频（共60分钟） 的课程，将英国精算师考试 CT1 和 CM1 的利息理论部分的历年真题归纳为14类题型，做了对应的知识点和真题精讲视频，一共12讲，总时长为60分钟30秒。

截个图：

笔画我调的有点粗，可以选个细一点的笔画。我录这一系列视频用的是很便宜的耳机，所以收音的效果还是用 ipad 直接录好一些。

classin 在备课教室进行本地录课后得到是 720P 的 flv 视频文件，本身就比较小，可以直接上传到教学平台里。非要说缺点的话，就是左下角会有 classin 的水印。但不影响观看。主要是方便，直接录好直接传到视频平台就行了。

课件支持所有通用格式，比如 PPT，上课前传到云盘即可。没记错的话，classin 是唯一一家能在直播时放 PPT 动画的教学直播平台，PPT里的声音也能听到：

我已经听到屏幕里传来的声音了，好听！好听就是好课。

因为 PPT 和 word 等上传至云盘需要转码以后才能使用，可能需要等待一段时间。所以如果老师们的 PPT 里没有动画，建议先转成 pdf 再上传，这样是无须转码就能用的。

方案四：电脑 + 数位板 + drawboard pdf + bandicam

这是我用 windows 电脑 + 数位板 + drawboard pdf 在pdf上进行书写的截图，录制视频可以用我前面讲的任意一款软件（ Bandicam，OBS，EV 录屏等）。macbook上，可以把这个方案里的 drawboard pdf 替换为 notability.

drawboard pdf 和 notability 是收费软件，但价格都在 100元 以内，从书写笔迹的体验来说，是值回票价的。

以上就是 Jackie 关于录制视频的全部心得啦，有帮助的话欢迎点个在看，同学们看到不妨转发给你们为了录课焦头烂额的老师们。开学快乐，哈哈~
