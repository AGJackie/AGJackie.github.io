【VBA】如何利用Class module计算精算现值

用 Excel VBA 中的类模块 Class module 计算精算现值。

本文背景：类模块 （Class module）的普及是未来VBA编程的发展趋势。它具有能够简洁化代码，增强代码可读性的特点。但同时，它会加大代码维护的难度。类模块 （Class module）相对门槛较高，比普通模块更难理解。考虑到做精算同事的背景，可能要两者权衡取舍。

不积跬步无以至千里，本文用了一个简单的类模块 （Class module）例子，帮助大家入门。本文设计了一个精算背景，目标读者为类模块初级学习者。欢迎大家指出不足之处。

我有一个朋友，她叫做小珂珂。

小珂珂是 ActuaryGarden 公司的精算实习生，这天，她在做精算表写VBA的时候遇到了一个问题：她手上目前有两套精算假设，想利用这两套假设计算年金现值a，还有定期寿险现值A。

因为不想重复工作，她想到，每一套假设都可以看作一个对象(Object)呀！可以利用类模块(Class model)来避免重复工作。
可以把利息率和死亡率设置成类模块(Class model)的属性，计算A和a设置成类模块(Class model)的方法~
